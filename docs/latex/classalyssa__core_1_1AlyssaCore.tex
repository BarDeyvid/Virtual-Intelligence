\doxysubsection{Detailed Description}
Class to manage language models and generate text. 

The \doxylink{classalyssa__core_1_1AlyssaCore}{Alyssa\+Core} class loads a base language model, manages its context, applies Lo\+RA adaptation if necessary, and generates text based on given prompts. It provides methods for getting model details and generating text.
